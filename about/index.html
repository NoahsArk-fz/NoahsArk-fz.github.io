恭喜你找到了“关于”页！如标题所见，这是一个属于你的网站。

想做一个这样的网站并不困难，也几乎没有任何成本，而它却可以永远存续下去，只要我们还有互联网，它就是一块自留地。

它的网址是： https://noahsark-fz.github.io

想理解这个网址的意思也并不困难，无非是诺亚方舟Noah’s Ark 和 FZ 而已，后面的后缀是因为它用了 Github 进行加载。

整个网站的一切包括文字、图片和框架都是可以自己调整设计的，它是全世界独一无二的，当然，目前看来这个网站只是静态的，只要不重新调整它就会一直以这样的状态待下去，无论何时何地。当然，现在这个网站已经转交给你，如果你想试试以后自己写一些图文或者其他内容放到它上面，我可以教给你应该怎么操作。

我愿意送给你所有你能想到和想不到的东西，我想这个网站属于其中之一。就像整个网页一打开的那张横屏图片一样，一个穿着红色连衣裙的小女孩站在一艘船前，远处是绵延的山，这是我对这个网站形象的一种想象，生活是模糊的深蓝色，但又是有光亮的。

《三体》里面，云天明送给程心一颗远处的恒星，而后，他就默默在太空中漂泊了一生。但云天明还是浪漫的，我想到的是书里面他送出星星后的一段描写：

云天明想，当程心看到这颗星时，自己已不在人世了。其实，他和程心看到的这颗星星，是它在二百八十六年前的样子，这束微弱的光线在太空中行走了近三个世纪才接触到他们的视网膜，而它现在发出的光线，要二百八十六年后才能到达地球，那时程心也不在人世了。

她将度过怎样的一生呢？但愿她能记得，茫茫星海中，有一颗星星是属于她的。

就和这个网站一样，它好像一颗漂在互联网上的星星。
